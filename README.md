# Gambitron Technologies OPC - Company Website

A modern, responsive website for Gambitron Technologies OPC, showcasing software development services including web development, mobile apps, and custom software solutions.

## 🚀 Features

- **Responsive Design**: Works perfectly on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, scroll animations, and smooth transitions
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Fast Loading**: Optimized CSS and JavaScript for performance

## 🚀 Deployment

### Option 1: Web Hosting
1. Upload all files to your web hosting service
2. Ensure `index.html` is in the root directory
3. Your website will be accessible at your domain

### Option 2: GitHub Pages
1. Create a new GitHub repository
2. Upload all files
3. Enable GitHub Pages in repository settings
4. Your website will be available at `username.github.io/repository-name`

### Option 3: Netlify
1. Drag and drop your project folder to Netlify
2. Get instant deployment with a free subdomain
3. Option to connect custom domain

## 🆘 Troubleshooting

### Common Issues
1. **Images not loading**: Check file paths and image URLs
2. **Styling issues**: Ensure CSS file is in the same directory
3. **JavaScript errors**: Check browser console for errors
4. **Mobile menu not working**: Verify JavaScript file is loaded
